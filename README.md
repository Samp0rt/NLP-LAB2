# Lab2

Этот репозиторий содержит ноутбук **Lab2.ipynb**. Для выполнения работы нужно просто последовательно запустить все ячейки ноутбука.

## Установка

1. Клонируйте репозиторий или скопируйте файлы к себе на машину.
2. Установите зависимости:

pip install -r requirements.txt

## Запуск ноутбука

1. Запустите через Jupyter Notebook:

jupyter notebook

2. В открывшемся интерфейсе перейдите к файлу `Lab2.ipynb`.
3. Откройте ноутбук и последовательно выполните все ячейки сверху вниз:
- Меню **Kernel → Restart & Run All**  
или  
- Нажимая **Run** для каждой ячейки по очереди.

Убедитесь, что в качестве ядра выбрано окружение, в котором были установлены зависимости (например, `lab2-env`).

## Запуск через VS Code

1. Откройте папку с проектом в **VS Code**.
2. Убедитесь, что установлены расширения:
- **Python**
- **Jupyter**
3. Откройте файл `Lab2.ipynb` прямо в VS Code.
4. Последовательно выполните все ячейки сверху вниз.
